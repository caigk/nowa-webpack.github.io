## nowa-github-pages

---

nowa github homepage

### Install

```shell
npm i gitbook nowa -g && nowa install
```

### Start

- start web
```shell
npm start
```

- start cli doc
```shell
npm run start_doc_cli
```

- start gui doc
```shell
npm run start_doc_gui
```

### Save

```shell
npm run save
```
