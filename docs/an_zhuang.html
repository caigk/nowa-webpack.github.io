安装

安装

准备

在安装 nowa 之前请先确保已经安装有 nodejs 和 npm。

强烈建议使用 nodejs>=4.0 && npm>=3.0 版本。

可以通过 node -v 和 npm -v 来查看 nodejs 和 npm 的版本。

安装

通过以下命令来安装 nowa 和常用插件：

npm i nowa -g
nowa install

如果提示无权限，请尝试加 sudo。
如果下载速度过慢，请增加 --registry https://registry.npm.taobao.org 参数
nowa install 会安装一批常用的插件。
