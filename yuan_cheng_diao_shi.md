# 远程调试

---

## 开发服务器的转发配置

`nowa server` 标配了远程调试的功能，该功能所做的事情是：将开发服务器中（指定路由下）找不到的资源转发到指定的服务器。例如有以下配置：

`abc.json`
```json
{
  "options": {
    "port": 3000,
    "proxy": "http://127.0.0.1:9077"
  }
}
```

那么对于 `http://127.0.0.1:3000/index.htm` 的访问，
- 如果开发服务器存在静态资源 `/index.htm`，则返回 `/index.htm`
- 如果开发服务器不存在静态资源 `/index.htm`，则返回 `http://127.0.0.1:9077/index.htm`
